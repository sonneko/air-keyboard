# Air Keyboard

[日本語はこちら](./README.ja.md)

## Overview
An application that tracks finger movements from the camera, enabling typing as if a physical keyboard were present.

## Technologies Used
- *MediaPipe Hand Landmarker*: Real-time hand tracking
- *Vite*: Build
- *Chart.js*: Chart rendering

## Demo
Experience it now in your browser [here](https://air-keyboard-sonneko.aoristos.net).
